# Smart Pointers

- [Item 18](./item18.md): Use `std::unique_ptr` for exclusive-ownership resource management.
- [Item 19](./item19.md): Use `std::shared_ptr` for shared-ownership resource management.
- [Item 20](./item20.md): Use `std::weak_ptr` for `std::shared_ptr`-like pointers that can dangle.
- [Item 21](./item21.md): Prefer `std::make_unique` and `std::make_shared` to direct use of `new`.
